# 📊 Sales & Inventory Insights Dashboard (Excel)

## 🔍 Project Overview
This project focuses on analysing retail sales data to generate meaningful insights for business decision-making. The raw dataset was cleaned, transformed, and analysed using Excel to build an interactive dashboard highlighting key performance metrics and trends.

---

## 🎯 Objective
To convert raw, unstructured sales data into actionable insights that support inventory 
optimisation, product performance analysis, and overall business strategy.

---

## 🛠️ Tools & Skills Used
- Microsoft Excel  
- Data Cleaning & Transformation  
- Pivot Tables & Pivot Charts  
- Dashboard Design  
- Data Analysis & Interpretation  

---

## 📊 Dashboard Features
- KPI metrics (Total Revenue, Total Sales, Top Product)  
- Monthly revenue trend analysis  
- Top-performing products identification  
- Category-wise sales distribution  
- Interactive filters using slicers  

---

## 📈 Key Insights
- A small group of top-performing products contributes a significant share of total revenue  
- Sales show clear trends across months, indicating demand patterns  
- Some products and categories consistently underperform  

---

## 💡 Business Recommendations
- Focus on high-performing products to maximise revenue  
- Improve strategy (pricing/marketing) for low-performing items  
- Optimise inventory planning based on demand trends
